# 04. Queue (큐)

연결 리스트 기반 FIFO 큐. `front`/`rear` 두 포인터로 양끝을 관리. "앞에서 빼서 뒤로 넣기"로 **원형 회전**을 구현해 요세푸스류 문제를 푼다.

## 구현 기능
- `push`(rear 삽입) / `pop`(front 삭제) — O(1)
- `front` / `size` / `empty`

## 파일별 문제
| 파일 | 문제 내용 |
|------|----------|
| p1.cpp | 기본 큐(`push/pop/front/size/empty`) + **sum_elements**: 큐 안 모든 값의 합 |
| p2.cpp | **요세푸스 변형** — 1..N 원형, 규칙 수열을 순환시키며 매번 그 수만큼 회전 후 한 명 제거. 마지막 **생존자 번호** 출력 |
| p3.cpp | p1의 변형 — **average**: 큐 원소들의 정수 평균 |
| p4.cpp | p2와 같은 제거 규칙에서 **특정 번호 X가 몇 번째로 제거되는지** 출력(끝까지 생존 시 -1) |

## 입력 형식 (p2 / p4)
```
T
N M [X]          (인원 수, 규칙 개수, [찾을 대상])
rule_1 ... rule_M
```
